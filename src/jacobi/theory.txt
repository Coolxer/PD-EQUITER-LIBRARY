# Zawartość: Opis teoretyczny i matematyczny dla metody Jacobiego.

------------------------------------------ Opis teoretyczny -------------------------------------------

Metoda iteracji prostej jak sama nazwa wskazuje jest jedną z najprostszych metod iteracyjnych
rozwiązywania układu równań.

-------------------------------------------------------------------------------------------------------

----------------------------------------- Opis matematyczny -------------------------------------------

Przekształcenie układu wejściowego:
    Ax = b
    (L + D + U)x = b
    Lx + Dx + Ux = b
    (L + U)x + Dx = b
    Dx = −(L + U)x + b
    x = −D^(−1)(L + U)x + D^(−1)b
    x = −D^(−1)[(L + U)x + b]

Wzór dla metody Jacobiego w postaci macierzowej:
    x^(k + 1) = D^(-1)*(b - (L + U)x^(k))

Wzór dla metody Jacobiego w postaci elementowej 
    x_i^(k + 1) = (1/a_ii)(b_i - sum(a_ij * x_j^(k)))

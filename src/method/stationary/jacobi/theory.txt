# Zawartość: Opis teoretyczny i matematyczny dla metody Jacobiego.

------------------------------------------ Opis teoretyczny -------------------------------------------

Metoda iteracji prostej jak sama nazwa wskazuje jest jedną z najprostszych metod iteracyjnych
rozwiązywania układu równań.

-------------------------------------------------------------------------------------------------------

----------------------------------------- Opis matematyczny -------------------------------------------

Przekształcenie układu wejściowego:
    A = L + D + U
    (L + D + U)x = b
    Lx + Dx + Ux = b
    Dx + (L + U)x = b       // - (L + U)x
    Dx = b - (L + U)x       // /D      ||     // * D^(-1)
    x^(n+1) = (b - (L + U)x^(n)) / D             ||     x^(n+1) = (b - (L + U)x^(n)) * D^(-1)

Wyznaczenie L + U na podstawie wzoru (możliwe jest także zwykłe obliczenie sumy macierzy L i U):
    A = L + D + U
    A = D + (L + U)
    (L + U) = A - D

Wzór dla metody Jacobiego w zapisie macierzowym (2 warianty):
    1) x^(n+1) = (b - (L + U)x^(n)) / D
    2) x^(n+1) = (b - (L + U)x^(n)) * D^(-1)
